# Design System: 01 - Color Palette

ชุดสีสำหรับ OrderUp Admin Dashboard ถูกออกแบบมาเพื่อให้ดูสะอาดตา เป็นมืออาชีพ และใช้งานง่าย โดยมีสีน้ำเงินเป็นสีหลัก (Primary) และสีเขียวเป็นสีรอง (Secondary) ตามที่กำหนด

## Primary Colors (สีหลัก)

ใช้สำหรับปุ่มหลัก, ลิงก์, และส่วนประกอบสำคัญที่ต้องการดึงดูดผู้ใช้

- **Primary (Blue):** `#007BFF` - สีน้ำเงินสดใสสำหรับ Call-to-action หลัก
- **Primary-Dark:** `#0056b3` - สำหรับสถานะ Hover หรือ Active
- **Primary-Light:** `#cce5ff` - สำหรับพื้นหลังของส่วนที่เกี่ยวข้อง

## Secondary Colors (สีรอง)

ใช้สำหรับองค์ประกอบรอง หรือส่วนที่ต้องการเน้นแต่ไม่สำคัญเท่าสีหลัก

- **Secondary (Green):** `#28A745` - สีเขียวที่สื่อถึงการเติบโต หรือใช้เป็นสี Success ได้ด้วย
- **Secondary-Dark:** `#1e7e34` - สำหรับสถานะ Hover
- **Secondary-Light:** `#d4edda` - สำหรับพื้นหลัง

## Neutral Colors (ชุดสีเทา)

เป็นกระดูกสันหลังของ UI ใช้สำหรับตัวอักษร, พื้นหลัง, เส้นขอบ, และสถานะ Disabled

- **Black:** `#121212` - สำหรับหัวข้อหลัก
- **Gray-800:** `#343A40` - สำหรับเนื้อหาทั่วไป
- **Gray-600:** `#6C757D` - สำหรับข้อความรอง, placeholder
- **Gray-400:** `#CED4DA` - สำหรับเส้นขอบ (borders)
- **Gray-200:** `#E9ECEF` - สำหรับพื้นหลังส่วนต่างๆ (panel background)
- **Gray-100:** `#F8F9FA` - สำหรับพื้นหลังหลักของหน้า
- **White:** `#FFFFFF` - สำหรับใช้บนพื้นหลังสีเข้ม

## Semantic Colors (สีตามความหมาย)

ใช้สำหรับสื่อสารสถานะต่างๆ ให้กับผู้ใช้ได้อย่างรวดเร็ว

- **Success:** `#28A745` (ใช้สีเดียวกับ Secondary) - สำเร็จ, ผ่าน
- **Warning:** `#FFC107` - แจ้งเตือน, ข้อควรระวัง
- **Danger:** `#DC3545` - ผิดพลาด, ลบ, อันตราย
- **Info:** `#17A2B8` - ข้อมูลทั่วไป
