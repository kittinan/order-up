# Design System: 02 - Typography

การออกแบบตัวอักษร (Typography) ที่ดีคือหัวใจของการสร้าง UI ที่อ่านง่ายและเป็นมิตรกับผู้ใช้ เราจะใช้ System Font Stack เพื่อประสิทธิภาพสูงสุดและความรู้สึกคุ้นเคยสำหรับผู้ใช้ในแต่ละระบบปฏิบัติการ

## Font Family (ชุดฟอนต์)

เราจะใช้ "System Font Stack" ซึ่งจะเลือกใช้ฟอนต์มาตรฐานของแต่ละ OS โดยอัตโนมัติ ทำให้ไม่ต้องดาวน์โหลดฟอนต์เพิ่มและโหลดหน้าเว็บได้เร็วขึ้น

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol";
```

## Type Scale (ขนาดตัวอักษร)

เรากำหนดขนาดตัวอักษรโดยใช้หน่วย `rem` เพื่อให้สามารถปรับขนาดได้ตามการตั้งค่าของผู้ใช้ และเพื่อความง่ายในการดูแลรักษาระยะห่าง (spacing) โดยมี `1rem` เท่ากับ `16px` (ขนาดพื้นฐานของ browser)

| Element | Font Size | Font Weight | Notes |
| :--- | :--- | :--- | :--- |
| `<h1>` | `2.25rem` (36px) | `700` (Bold) | สำหรับหัวข้อหลักของหน้า |
| `<h2>` | `1.875rem` (30px) | `700` (Bold) | สำหรับหัวข้อรอง |
| `<h3>` | `1.5rem` (24px) | `700` (Bold) | สำหรับหัวข้อย่อยใน Section |
| `<h4>` | `1.25rem` (20px) | `700` (Bold) | สำหรับหัวข้อใน Card/Panel |
| `<p>` (Body) | `1rem` (16px) | `400` (Normal) | สำหรับเนื้อหาทั่วไป |
| `<small>` (Sub-text) | `0.875rem` (14px) | `400` (Normal) | สำหรับคำอธิบายเล็กๆ, meta data |
| `label` | `1rem` (16px) | `600` (Semi-bold) | สำหรับป้ายกำกับฟอร์ม |

## Font Weights (ความหนา)

เพื่อความเรียบง่ายและชัดเจน เราจะใช้ความหนาหลักๆ เพียง 2-3 ระดับ

- **Normal:** `400` - สำหรับเนื้อหาทั่วไป (Body text)
- **Semi-bold:** `600` - สำหรับ Labels หรือข้อความที่ต้องการเน้นเล็กน้อย
- **Bold:** `700` - สำหรับหัวข้อ (`<h1>` - `<h4>`) และข้อความที่ต้องการเน้นเป็นพิเศษ
